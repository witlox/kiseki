# RFC 1057 §9.2 — AUTH_SYS / AUTH_UNIX trivial root credential fixture

## Source
RFC 1057 §9.2 (RPC: Remote Procedure Call Protocol Specification
Version 2, June 1988):
  https://www.rfc-editor.org/rfc/rfc1057#section-9.2

The XDR grammar:
  struct authsys_parms {
      unsigned int stamp;
      string       machinename<255>;
      unsigned int uid;
      unsigned int gid;
      unsigned int gids<16>;
  };

This fixture is the trivial root credential discussed throughout
the RFC text:
  stamp        = 0
  machinename  = "unix" (4 bytes; XDR pads to 4-byte boundary so no pad bytes)
  uid          = 0 (root)
  gid          = 0 (root)
  gids         = []  (empty supplemental gids array)

  Total = 24 bytes (4 + 4 + 4 + 4 + 4 + 4).
